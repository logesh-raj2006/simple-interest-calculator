# simple-interest-calculator
## Description
This project explains how to calculate **Simple Interest** using a basic mathematical formula.  
Simple Interest is commonly used in banking and finance to calculate interest on loans and deposits.


## Formula
Simple Interest (SI) is calculated using the formula:

SI = (P × R × T) / 100

Where:
P = Principal amount  
R = Rate of interest (per annum)  
T = Time period (in years)


## Example Calculation
Let:
- Principal (P) = 10,000  
- Rate (R) = 5%  
- Time (T) = 2 years  

SI = (10,000 × 5 × 2) / 100  
SI = 1,000  


## Use Case
This calculator helps beginners understand how interest is calculated in:
- Savings
- Loans
- Fixed deposits
